== Description ==

Adds oEmbed support for [AllGrain.Beer](http://allgrain.beer/ "Brew better beer!")
Try adding a recipe URL to your post or page!

== Installation ==

1. Upload `allgrainbeer` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it

== Frequently Asked Questions ==

= What is AllGrain.Beer? =

A beer recipe calculator. [Give it a try!](http://allgrain.beer/)

= How do I embed a recipe? =

Simply add the recipe URL to a post/page. For instance: `http://allgrain.beer/recipes/914/`
